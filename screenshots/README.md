## วิธีใช้ (ใช้ Edge/Chrome ที่มีในเครื่องอยู่แล้ว — ไม่ต้องโหลดเบราว์เซอร์เพิ่ม)

```powershell
cd "D:\Users\zen\project test 1\screenshots"
npm install
node capture.mjs
```

> ℹ️ **สำคัญ:** รูปจะเกิดขึ้น **หลังรัน `node capture.mjs`** เท่านั้น — ถ้ายังไม่รัน โฟลเดอร์จะว่าง
> เครื่องต้องมี Microsoft Edge หรือ Google Chrome (Windows 11 มี Edge อยู่แล้ว)

ระหว่างรันจะเห็น `✓ light 01-login`, `✓ dark 09-admin` ... ไล่ไปเรื่อย ๆ
เสร็จแล้วรูปอยู่ใน:
- `screenshots/light/` — ธีมสว่าง
- `screenshots/dark/`  — ธีมมืด

ถ้าบรรทัดไหนขึ้น `✗` = หน้านั้นแคปพลาด (ข้ามไป) — ลองรันใหม่ หรือเพิ่มเวลา `waitForTimeout`

## ปรับแต่ง

- ถ้าหน้าไหนแคปพลาด (timeout) สคริปต์จะข้ามและทำหน้าอื่นต่อ แล้วขึ้น `✗` — ลองเพิ่มเวลา `waitForTimeout` ในขั้นตอนนั้น
